Slider - 3 points

En haut de la page, créer un slider grâce à la librairie swiper. Il contiendra 3 slides avec une image de fond, et à chaque fois, un titre et un petit texte de votre choix mettant en avant le JPO (visites, ateliers…). Le titre et texte seront centrés verticalement et horizontalement sur le slide grâce à du flex.

Étape 1 : Mise en place du HTML

          
            <div id="slider-container">
            <div id="slider">
                <div class="slide" ><img src="images/1.jpg" alt=""></div>
                <div class="slide" ><img src="images/2.jpg" alt=""></div>
                <div class="slide" ><img src="images/3.jpg" alt=""></div>
                <div class="slide" ><img src="images/4.jpg" alt=""></div>
            </div>
            <div id="prev">&lt;</div>
            <div id="next">&gt;</div>
        </div>

Étape 3 : Ajout de la logique JavaScript

Passons maintenant à la partie JavaScript. Nous allons utiliser JavaScript pour rendre notre slider interactif en permettant la navigation entre les diapositives.

    
      const slider = document.getElementById('slider');
    const prevButton = document.getElementById('prev');
    const nextButton = document.getElementById('next');

    let currentIndex = 0;

    function updateSlider() {
        const newPosition = -currentIndex * 100 + '%';
        slider.style.transform = 'translateX(' + newPosition + ')';
    }

    function nextSlide() {
        currentIndex = (currentIndex + 1) % slider.children.length;
        updateSlider();
    }

    function prevSlide() {
        currentIndex = (currentIndex - 1 + slider.children.length) % slider.children.length;
        updateSlider();
    }

    nextButton.addEventListener('click', nextSlide);
    prevButton.addEventListener('click', prevSlide);

Étape 3 : Ajout de la logique JavaScript

Ajoutons maintenant un peu de logique JavaScript pour rendre notre formulaire interactif. Nous allons écouter l'événement de soumission du formulaire et afficher les données saisies dans la console pour l'instant.

    
      const tabs = document.querySelectorAll('.tab');
    const contents = document.querySelectorAll('.content');

    tabs.forEach(tab => {
        tab.addEventListener('click', function() {
            tabs.forEach(t => t.classList.remove('tab-active'));
            contents.forEach(c => c.classList.remove('active'));

            this.classList.add('tab-active');
            
            const axeClass = this.classList[1]; // Utilisez la deuxième classe pour l'axe (JeuxVidéo, CommunicationDigitale, etc.)
            const content = document.querySelector(`.content.${axeClass}`);
            content.classList.add('active');
        });
    });
